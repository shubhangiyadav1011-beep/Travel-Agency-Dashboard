# ✈️ Travel Agency Performance Dashboard

> 📊 An interactive Power BI dashboard designed to analyze travel agency performance through revenue, profitability, customer insights, regional analysis, and operational KPIs.

---

## 📖 Overview

The **Travel Agency Performance Dashboard** is a Business Intelligence project built using **Microsoft Power BI**. It transforms raw travel booking data into meaningful visualizations that help monitor business performance, identify trends, and support data-driven decision-making.

The dashboard provides an end-to-end analysis of revenue, profit, customer behavior, regional performance, payment methods, and operational efficiency through interactive reports.

---

## 🎯 Project Objective

The objective of this project is to:

- 📈 Monitor overall business performance
- 💰 Analyze revenue and profit trends
- 🌍 Compare regional performance
- 👥 Understand customer behavior
- 💳 Analyze payment preferences
- 📦 Evaluate operational efficiency
- 📊 Support strategic business decisions
